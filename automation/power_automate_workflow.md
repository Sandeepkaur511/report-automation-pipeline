# Power Automate Workflow

## Scheduled Trigger

The workflow runs once daily after the Power BI dataset refresh.

Schedule example:

Dataset Refresh → 09:00 AM
Automation Flow → 10:00 AM

---

## Steps

1. Trigger flow based on schedule
2. Retrieve report dataset from Power BI
3. Write output data to Excel Online
4. Apply formatting logic
5. Generate final report structure
6. Send email notification with report output
